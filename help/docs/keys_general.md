# General keys and their functions

Numlock must be on

| Function                  | Mouse | Keyboard  | Comment                                                            |
|---------------------------|-------|-----------|--------------------------------------------------------------------|
| All clear                 | AC    | a         | Optional: double-click Numlock (must be enabled in Tools/Options) |
| Arithmetic calculations   | +     | +         | On main Keyboard and Numpad |
|                           | -     | -         | On main Keyboard and Numpad |
|                           | *     | *         | On main Keyboard and Numpad |
|                           | /     | /         | On main Keyboard and Numpad |
|                           | =     | Enter     | On main Keyboard and Numpad |
| Backspace                 | <-    | Backspace | Optional on Numpad: Shift + . or Shift + , depending on local settings |
| Clear (operations retain) | C     | c         | Optional: Shift + Numlock (must be enabled in Tools/Options) |
| Decimal separator         | .     | .         | On main Keyboard and Numpad. Depending on local settings, see also Tools/Options for replacing/converting |
|                           | ,     | ,         | On main Keyboard and Numpad. Depending on local settings, see also Tools/Options for replacing/converting |
| Delete last line from history | DL | Shift + DL | On main Keyboard and Numpad |
| Delete last operation     | DO    | Ctrl + DO | On main Keyboard and Numpad |
| Exponent                  | EXP   | e or E    |                             |
| Number input              | 0 - 9 | 0 - 9     | On main Keyboard and Numpad |
| Sign insert               | (-)   | _ (underscore) | Shift + Numpad Minus |
