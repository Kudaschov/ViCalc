# Number Input Format

**Format:**  
-1.23456789e-123 (English/US)  
-1,23456789e-123 (German/European)  

**Rules:**<br>
- **Negative sign:** `_` (underscore) or Shift + Numpad Minus  
  *(Regular Minus = subtraction)*  
- **Decimal separator:** `.` or `,` depending on locale  
- **Exponent:** `e` or `E`
