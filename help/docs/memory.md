# Memory keys and their functions

Numlock must be on

| Function                  | Label | Comment                                                    |
|---------------------------|-------|------------------------------------------------------------|
| Memory division           | M/    |                                                            |
| Memory exchange           | X<>M  |                                                            |
| Memory minus              | M-    | On main Keyboard and Numpad                                |
| Memory multiplication     | M*    |                                                            |
| Memory plus               | M+    | On main Keyboard and Numpad                                |
| Memory recall             | MR    | On main Keyboard and Numpad                                |
| Memory store              | MS    | On main Keyboard and Numpad                                |
